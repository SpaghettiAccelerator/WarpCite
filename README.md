# WarpCite

Write academic Typst documents with citations you can actually verify. WarpCite
OCRs your source PDFs, generates BibTeX for them, and lets you preview every
citation right where you write it.

## Features

- **Document OCR** — right-click a PDF → *Run Document OCR*. The result is
  stored as `<name>.ocr.json` next to the PDF, including per-page markdown,
  headers/footers and the printed page number (arabic or roman) parsed from
  common footer formats.
- **BibTeX generation** — right-click a PDF → *Generate BibTex*. WarpCite finds
  an arXiv ID or DOI (filename, text layer, metadata) and fetches the canonical
  entry; with your consent it falls back to extracting the metadata via OCR,
  or creates a TODO stub.
- **Automatic aggregation** — all `sources/*/source.bib` files are merged into
  a `sources.bib` between autogenerated markers. Your own entries above the
  markers are preserved.
- **Citation preview** — hover a `#dhbwCite("source", page: 3, highlight: (10, 80))`
  call to see the cited page rendered with the cited passage highlighted.
  Cmd+click opens the full document in a side panel, scrolled to the citation.
  Images can be exported as PNG from the panel.
- **Agent-ready** — one click in the settings copies instructions that teach an
  AI coding agent to research exclusively from your OCR'd sources and cite with
  precise page and highlight references.

## Quick start

1. Set your Mistral API key: run **WarpCite: Set Mistral API Key** from the
   command palette. The key is kept in VS Code secret storage, not in your
   settings files.
2. Put each source PDF in its own folder: `sources/<source-name>/paper.pdf`.
3. Right-click the PDF: *Run Document OCR*, then *Generate BibTex*.
4. Cite in your `.typ` file and hover the call to verify the passage:

```typst
#dhbwCite("<source-name>", page: 3, highlight: (120, 240))
```

## Settings

| Setting | Default | Description |
| --- | --- | --- |
| `warpcite.mistral.model` | `mistral-ocr-latest` | OCR model |
| `warpcite.citationFunction` | `dhbwCite` | Typst citation function recognized for hover and Cmd+click |

## Requirements

- A [Mistral API key](https://console.mistral.ai) for OCR and OCR-based BibTeX
  extraction. Identifier-based BibTeX (arXiv/DOI) and citation previews work
  without it.
